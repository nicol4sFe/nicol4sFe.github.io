CSS Grid Layout

Comdori, N. Y. (n.d.). css grid names for line numbers Archives. NY Comdori Computer Science Note. Retrieved October 23, 2022, from https://nycomdorics.com/tag/css-grid-names-for-line-numbers/

CSS Grid Layout (también conocido como "Grid" o "CSS Grid"), es un sistema de diseño bidimensional basado en cuadrículas que, en comparación con cualquier sistema de diseño web del pasado, cambia por completo la forma en que diseñamos las interfaces de usuario. CSS siempre se ha utilizado para diseñar nuestras páginas web, pero nunca ha hecho un buen trabajo. Primero, usamos tablas, luego flotadores, posicionamiento y bloques en línea, pero todos estos métodos eran esencialmente trucos y dejaban fuera muchas funciones importantes (centrado vertical, por ejemplo). Flexbox también es una excelente herramienta de diseño, pero su flujo unidireccional tiene diferentes casos de uso, ¡y en realidad funcionan bastante bien juntos! Grid es el primer módulo CSS creado específicamente para resolver los problemas de diseño que todos hemos estado solucionando desde que creamos sitios web.

Su CSS puede colocarlos en cualquier orden, lo que hace que sea muy fácil reorganizar su cuadrícula con consultas de medios. Imagine definir el diseño de toda su página y luego reorganizarla por completo para acomodar un ancho de pantalla diferente, todo con solo un par de líneas de CSS. Grid es uno de los módulos CSS más potentes jamás introducidos.

Arquitectura de cuadrícula CSS

Hay dos formas de usar diseños de cuadrícula CSS: implícita y explícita. Con una cuadrícula implícita, simplemente defina el número de columnas que desea y el navegador creará la cuadrícula automáticamente. Con una cuadrícula explícita, se define tanto el número de columnas como el número de filas. Esto le da más control sobre el diseño, pero puede ser más complicado de configurar.

Es un sistema de diseño bidimensional. Esto significa que puede manejar columnas y filas. Sin embargo, a diferencia de los diseños CSS tradicionales, que son en su mayoría unidimensionales, CSS Grid está diseñado para trabajar con ambas dimensiones al mismo tiempo. Es un sistema de diseño basado en contenedores. Esto significa que se aplica a los elementos que son hijos de un elemento contenedor. El elemento contenedor define la cuadrícula y los elementos secundarios se colocan en las celdas de la cuadrícula.

Es un sistema de diseño receptivo. Esto significa que puede adaptarse a diferentes tamaños de pantalla y resoluciones. CSS Grid también es flexible, lo que significa que se puede utilizar para una amplia variedad de diseños, desde simples hasta complejos.

Se construye sobre el modelo de caja CSS existente. Esto implica que se puede utilizar con cualquier base de código CSS existente. Sin embargo, también tiene algunas características nuevas que lo hacen único.

Características que hacen que CSS Grid sea único

Proporciona la capacidad de colocar elementos en la cuadrícula mediante el posicionamiento basado en líneas. Esto permite crear diseños muy complejos sin tener que utilizar flotadores tradicionales o posicionamiento absoluto.

Proporciona la capacidad de abarcar columnas y filas. En otras palabras, puede tener elementos que abarquen varias columnas o filas.

Proporciona la capacidad de colocar elementos en ubicaciones específicas mediante números de línea y nombres, o dirigiéndose a un área específica de la cuadrícula. También se incluye un algoritmo para controlar la colocación de elementos que no se colocan explícitamente en la cuadrícula.

Proporciona la capacidad de controlar cómo se alinean los elementos una vez colocados dentro del área de la cuadrícula y cómo se alinea la cuadrícula en general.

Proporciona la capacidad de crear una cuadrícula utilizando tamaños de pista fijos mediante píxeles, lo que establece la cuadrícula en el píxel especificado que se ajusta al diseño que desea. También puede especificar tamaños de pista flexibles utilizando términos de porcentaje o la unidad.

Beneficios de usar CSS Grid

Hay muchos beneficios al usar CSS Grid al crear páginas web. Uno de los principales beneficios es que hace que sea mucho más fácil crear diseños complejos. Con CSS Grid, puede crear diseños con varias columnas y filas, y puede controlar fácilmente el tamaño y la posición de cada elemento en la página.

Otro beneficio de CSS Grid es que ayuda a mantener su código limpio y organizado. Con CSS tradicional, puede ser fácil que su código se vuelva desordenado y difícil de leer. Sin embargo, con CSS Grid, todos sus estilos se aplican a una cuadrícula, lo que hace que su código sea mucho más fácil de leer y entender.
