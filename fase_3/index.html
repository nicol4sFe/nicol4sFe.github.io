Etiquetas Semanticas

Las etiquetas semánticas hacen parte de HTML5 y estas etiquetas son nombres especiales que informan sobre el tipo de contenido que va dentro de las etiquetas. Esto crea una un orden y una compresión en el código, además de instruir al navegador sobre como indexar el contenido del sitio. Dentro del etiquetado semantico, hay diferentes etiquetas, las mas elementales para construir un esquema principal, serian el header, el article y el footer. Estas etiquetas no tienen estilos, ellas solo organizan la estructura del sitio web, mas no posicionan los textos, ni dan color a ellos, ni tamaños, esa parte de estilos se encarga CSS.

header: Esta se llama cabecera en español y va en la parte superior de la estructura.
nav: Esta etiqueta separa un apartado de enlaces con el fin de organizarlos como un menu y a su vez con submenu.
main: Como su nombre indica, esta etiqueta muestra el contenido principal de un documento.
section: Esta etiqueta define el contenido como una seccion del documento.
article: Esta etiqueta define el contenido en un articulo del documento.
aside: Esta etiqueta define el contenido en la parte lateral de la estructura.
footer: Esta etiqueta hace referencia al pie de la pagina o seccion.

Pardo, S. (2020, October 26). Etiquetas semánticas de HTML5. DEV Community 👩‍💻👨‍💻. https://dev.to/sapardo/etiquetas-semanticas-de-html5-3j7m

CSS Grid

Se puede definir como la division de una pagina en regiones principales y la division en relacion con el tamaño, posiciones y capas, y esto se realiza en el documento html como estilos. Grid en español significa cuadricula, entonces por eso el termino division de pagina, porque una cuadricula es un conjunto de lineas horizontales y verticales que se cruzan y forman las columnas y filas, donde los elementos html se colocan dentro de esta cuadricula y luego se le aplican diseños de estilo a la cuadricula.

column-gap: Esta propiedad especifica el espacio entre columnas y filas.
gap: Una propiedad abreviada para las propiedades de espacio entre filas y columnas.
grid: Propiedad abreviada para las propiedades grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow.
grid-area: Especifica un nombre para el elemento de la cuadricula, o esta propiedad es una propiedad abreviada para las propiedades grid-rows-start, grid-columns-start, grid-rows-end, grid-columns-end.
grid-auto-columns: Especifica el tamaño de columnas predeterminado.
grid-auto-flow: Especifica como se insertan los elementos colocados automaticamente en la cuadricula.

Hampton-Smith, S. (2016). CSS Grid Layout. In Pro CSS3 Layout Techniques (pp. 103–128). Apress.
